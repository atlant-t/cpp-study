# Learning C++

It's just my sandbox for learning c++ language.

## Dependencies

- **g++** 9.2.0 or more
- **make** 4.2.1 or more

## Questions

> Can I to open in MS Visual Studio?

Try using [File]->[New]->[Project From Existing Code...] and select the lesson folder.
